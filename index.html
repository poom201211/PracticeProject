House Simulation


Welcome Home
This is a house stimulation we have put together. What this website does is that it connects with the hardware Adruino and NodeMCU. The interesting thing is that we can control the functionality through just a press of a button. The hardware we used in this project includes, LED(Blue for air), LED(Green for light), ultrasonic and buzzer.

The bell button located below allows the user to trun on/off the buzzer which is located in the prototype house.
The air button below activates the LED(Will be in blue since we do not have a fan) which is located in the house aswell.
The door activates the ultrsonic sensor which stimulates the movement of someone entering the door or if there isnt anyone.
The light button activates LED which is again, located in the house. Meant to stimulate light source in the house with an LDR.
